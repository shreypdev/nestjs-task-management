## Description

An efficient and scalable server-side applications to manage tasks. It is build using [Nest](https://github.com/nestjs/nest) a progressive [Node.js](http://nodejs.org) framework with TypeScript.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
